# CaLab

**Calcium imaging analysis tools — deconvolution and data preparation.**

CaLab provides fast FISTA-based deconvolution (powered by a Rust solver), interactive parameter tuning via a browser UI, and utilities for loading data from CaImAn and Minian pipelines.

```bash
pip install calab
```
